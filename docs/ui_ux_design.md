# VOC Truck Tracking UI/UX Design Brief

## Product Feel

VOC Truck Tracking should feel like a modern logistics control room: clear, fast, trustworthy, and easy to scan during daily operations. The interface should prioritize status visibility, action clarity, and confidence for both admin users and drivers.

## Design Direction

- Use a stronger Material 3 theme with a logistics-inspired palette: deep navy, safety orange, success green, warning amber, and clean neutral surfaces.
- Replace plain grids and basic cards with structured dashboard sections, status-focused cards, and clear action tiles.
- Use compact rounded surfaces, readable typography, and consistent spacing so the app feels professional on mobile screens.
- Keep workflows direct: login, overview, monitoring, and trip actions should be visible without visual clutter.

## Key Screen Improvements

- Login: branded welcome panel, clear credential fields, full-width primary action, and role-aware copy.
- Admin dashboard: operational header, quick status metrics, and clearly grouped management actions.
- Driver dashboard: personalized shift summary, active trip status, truck/destination/cargo cards, and prominent trip actions.
- Cards: consistent elevation, icons, status colors, and spacing across summary, trip, and truck components.

## Visual System

- Primary: deep navy for navigation and important structure.
- Accent: orange for primary logistics actions.
- Success: green for active/online states.
- Warning: amber/orange for alerts and attention states.
- Surface: off-white app background with white cards for contrast.
- Shape: 8px to 16px radius depending on component importance.

## UX Principles

- Make important numbers and statuses readable at a glance.
- Show icon plus text for operational actions.
- Avoid decorative layouts that slow repeated use.
- Keep buttons large enough for field usage.
- Use responsive layouts so dashboard tiles do not feel cramped on small devices.
